## Visão geral

Este portfólio foi criado para organizar minha presença profissional como front-end, mostrando quem sou, quais tecnologias estudo, quais projetos desenvolvi e como minha experiência com design ajuda na construção de interfaces.

A proposta é apresentar projetos de forma clara para recrutadores, clientes e outros desenvolvedores, com foco em responsividade, estética e comunicação profissional.

---

## Objetivo profissional

Busco minha primeira oportunidade como **Desenvolvedor Front-end Júnior**, contribuindo com:

- Interfaces modernas e responsivas
- Componentização com React
- Organização visual e técnica de projetos
- Cuidado com UI, UX e acabamento
- Vontade real de aprender e evoluir em equipe

## Como rodar localmente

```bash
git clone https://github.com/WessYu/Portfolio---React.git
cd Portfolio---React
npm install
npm run dev
```

Depois acesse no navegador:

```txt
http://localhost:5173
```
